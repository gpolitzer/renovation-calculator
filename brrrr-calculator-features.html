Key Features
🧮
Formula Panels
Every line item has an expandable panel showing how its cost is calculated. Parameters are editable inline — change the sqft rate, labor cost, or quantity and the total updates instantly.
Adjustable per-item params
📐
Sqft & Bath-Count Scaling
Roofing, painting, plumbing, and flooring costs auto-scale from the property's sqft and bathroom count. Change sqft once at the top; all formulas update.
e.g. Roof = SqFt × $7
🔁
Toggle Include / Exclude
Each row has a YES/NO toggle. Excluded items are visually dimmed and removed from the total — useful for scoping different renovation scenarios.
Per-row on/off switch
📜
Permits Section
Manual permit entry with date and notes fields. Live integration with Jacksonville's EPICS API — one click fetches all city permits on file for the property address.
JAX EPICS API live fetch
💾
Save & Compare Projects
Export the current state as a self-contained HTML file. Re-open saved projects later. Use Compare Projects to view multiple properties side by side.
HTML self-contained export
⚙️
Edit Calculator Defaults
Toggle Edit Calculator Mode to adjust formula parameters (unit rates, base costs) and save them as new defaults — persisted in localStorage for all future projects.
localStorage persistence
📝
Notes per Line Item
Each formula panel includes a freeform notes textarea — document contractor quotes, site observations, or scope decisions alongside the cost estimate.
In formula panel
➕
Custom Categories & Rows
Add entirely new categories with custom names, or add custom rows inside existing categories. Each custom row has editable materials, labor, and total cost fields.
Fully extensible
🔍
Termite Decision Logic
The termite item is a full calculator in itself: choose Sentricon vs Termidor, optionally add tent fumigation (for active Drywood infestations), shows price ranges by house size bracket.
Sentricon · Termidor · Tent

Typical Workflow
Evaluating a New Deal
  1. Enter property address, sqft, beds, baths
  2. Toggle ON the items likely to need work
  3. Expand formula panels to adjust cost assumptions
  4. Click Fetch City Permits to see permit history
  5. Review the Total Renovation Cost summary
  6. Save Project as HTML with the property address
Comparing Multiple Properties
  1. Save each property as its own project file
  2. Open Compare Projects
  3. Load 2–3 saved project files side by side
  4. Review total costs, category breakdowns, notes
  5. Identify the deal with best reno-to-value ratio
  6. Use Edit Calculator mode to lock in local pricing
